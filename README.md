# 🐾 PawFocus-学习伙伴

一款常驻桌面的番茄钟应用，搭配像素风宠物陪伴。专注学习赚取金币，解锁更多宠物伙伴。

![主界面](docs/screenshots/main.png)

---

## 功能

### 番茄钟

- 倒计时 / 正计时两种模式
- 开始、暂停、恢复、重置
- 自动完成和手动提前完成
- 短休息 / 长休息循环
- 迷你悬浮窗，主窗口隐藏时仍可见

![迷你悬浮窗](docs/screenshots/mini-timer.png)

### 金币系统

- 完成专注获得金币奖励
- 不暂停的完美专注有额外加成
- 每完成 3 次触发连击奖励
- 金币数在托盘 tooltip 中实时显示

### 宠物系统

- 首次使用选择小猫或小狗
- 宠物商店可用金币购买另一只宠物
- 已拥有的宠物可以改名（100 金币）和切换
- 宠物根据时段切换不同外观色调
- 点击宠物弹出对话气泡

### 习惯打卡

- 创建习惯，支持每日 / 工作日 / 每 N 天 / 指定星期几
- 今日一键打卡和取消打卡
- 日历视图查看和编辑历史记录
- 编辑习惯名称、备注和频率

### 备忘便签

- 添加备忘，支持分类标签和截止日期
- 拖拽排序，双击编辑
- 按分类筛选

### 统计

- 今日 / 累计专注时长
- 本周每日专注时长柱状图
- 连续打卡天数

### 设置

- 调整工作时长、休息时长
- 开关音效、通知、健康提醒
- 设置几次专注后触发长休息

### 其他

- 系统托盘常驻，左键显示 / 隐藏
- 健康提醒（喝水、护眼、起身活动）
- 番茄钟完成系统通知
- 本地数据持久化存储

## 开发

### 环境要求

- Node.js 18+
- Rust

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run tauri dev
```

### 构建

```bash
npm run tauri build
```

构建后的安装包位于 `src-tauri/target/release/bundle/nsis/`。
